# DocuFlow-AI

AI-powered legal document automation platform. Upload a template document, answer questions conversationally, and generate completed documents.

## Stack

**Backend:** Go (chi router, unioffice for .docx, OpenAI API)
**Frontend:** React (Vite + TypeScript)
**Storage:** In-memory sessions
**Hosting:** Render (backend) + Vercel (frontend)

## Features

- Upload .docx templates with `{{placeholders}}`
- AI-powered conversational form filling
- Document generation and download
- Clean, modern UI
